Game Guide

This is a two player game where you, the player, gets to go first every time. You are given a list of even number of values, which are generated randomly. Every turn, you must choose between the first, or last value. The player with the largest sum of values wins at the end.

Controls: just click/tap either the first or last number that isn't already highlighted! The reset button will completely restart the game. The game ends when all numbers have been collected.

The goal is to develop a strategy where the first player, you, will win every time no matter the generated values. There is a strategy/algorithm, that you can use to win every time once you go first.
  • This strategy can be described in terms of 3 steps. Think about how you can prepare a method before you begin selecting. 1
  • Try to find patterns when you win. Don't just reset the game everytime, watch the colors and keep in mind what you selected in a win or loss. 2

  • An array is used here to store a collection of numbers. Each element in an array is identified by an index. In this case the index of the first number is 0, the next is 1, and so on until the last element/number.
